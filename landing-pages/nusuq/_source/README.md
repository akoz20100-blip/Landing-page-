# نُسق — Nusuq

> "حيث تعانق القهوة سحر الطبيعة الاستوائية."

علامة تجارية تدمج بين الضيافة والزراعة: مقهى ومشتل نباتي بتصميم مستوحى من
الجزر الاستوائية. تجمع بين مساحات عمل هادئة وأجواء استرخاء عميقة، مع تجربة
تخصيص فخاريات وزراعة نباتات للعملاء. التفاصيل الكاملة في
[بايو أثير](../../atheer/_source/BIO.md#نُسق).
